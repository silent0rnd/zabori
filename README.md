# ЕСС - заборы под ключ

Статический сайт на React и Vite для производства и монтажа заборов в Екатеринбурге.

## Локальный запуск

Требуется Node.js 20 или новее.

```bash
npm install
npm run dev
```

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите в него содержимое этой папки.
2. Откройте **Settings - Pages** и в разделе **Build and deployment** выберите **GitHub Actions**.
3. После отправки изменений в ветку `main` сайт соберётся и будет опубликован автоматически.

Для проверки сборки до публикации используйте:

```bash
npm run build
```
